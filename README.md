# Employee-Payroll
# 💼 Employee Payroll System

A full-stack web application to manage employee salary records, calculate payroll, and perform CRUD operations.

---

## 🚀 Live Demo

🌐 Frontend: https://employee-payrol.netlify.app/
⚙️ Backend: https://employee-payroll-1msv.onrender.com

---

## 📌 Features

* ➕ Add new employees
* 📋 View employee list
* ✏️ Update employee details
* ❌ Delete employee
* 💰 Automatic salary calculation
* 📊 Clean dashboard UI

## ⚙️ Installation & Setup (Run Locally)

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/employee-payroll.git
cd employee-payroll
```

### 2️⃣ Setup Backend

```
cd backend
npm install
node server.js
```

Backend runs on:

```
http://localhost:3000
```

### 3️⃣ Run Frontend

Open:

```
fontend/index.html
```

OR use Live Server.

## 🔗 API Endpoints

| Method | Endpoint       | Description       |
| ------ | -------------- | ----------------- |
| GET    | /employees     | Get all employees |
| POST   | /employees     | Add new employee  |
| PUT    | /employees/:id | Update employee   |
| DELETE | /employees/:id | Delete employee   |
